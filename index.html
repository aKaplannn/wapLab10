Lab 10 Solution 1

Array.prototype.even = function () { return this.filter(n => n % 2 == 0); }
Array.prototype.odd = function () { return this.filter(n => n % 2 !== 0); }
console.log([1, 2, 3, 4, 5, 6, 7, 8].even());
console.log([1, 2, 3, 4, 5, 6, 7, 8].odd());

Lab 10 Solution 2 - 2

Callback of process.nextTick gets executed multiple chances in one iteration, highest priority.
setImmediate only gets executed once in one cycle.
process.nextTick(); all of callbacks in nextTick queue, drain out.
Don't use while true loop on process.nextTick, it blocks event loop.
setImmediate; certain, the remaining for next iteration/tick
process.nextTick; API provided natively by Node.js
setImmediate is provided by libuv.
